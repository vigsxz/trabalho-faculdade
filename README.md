# Barbearia do João

## Sobre o projeto

Este projeto foi desenvolvido como trabalho da disciplina de Desenvolvimento Front-end.

O objetivo do site é apresentar os serviços de uma barbearia e facilitar o contato com clientes.

## Funcionalidades

* Exibição de serviços oferecidos
* Layout responsivo para celular e desktop
* Botão interativo de contato utilizando JavaScript

## Tecnologias utilizadas

* HTML
* CSS
* JavaScript
* Bootstrap
