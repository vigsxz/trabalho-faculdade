# Trabalho de Desenvolvimento Front-end

## Nome do projeto

Barbearia do João

## Integrantes

* Vitor

---

## 1. Descrição do caso

O projeto foi desenvolvido para um barbeiro autônomo que não possui presença digital estruturada.

---

## 2. Necessidade identificada

Foi identificada a dificuldade de divulgação dos serviços e contato com clientes.

---

## 3. Público-alvo

Homens jovens e adultos que buscam serviços de corte de cabelo e barba.

---

## 4. Objetivo do site

Permitir que o usuário visualize os serviços e entre em contato para agendar um horário.

---

## 5. Diagnóstico inicial

O cliente não possui site e depende apenas de indicações. O público precisa acessar informações básicas de forma rápida.

---

## 6. Processo de desenvolvimento

O projeto foi desenvolvido em etapas:

* planejamento do conteúdo
* estruturação em HTML
* estilização com CSS
* implementação de JavaScript
* testes de responsividade

---

## 7. Principais decisões

* uso de HTML semântico
* uso de Bootstrap para responsividade
* layout simples e direto
* uso de imagens para melhorar comunicação

---

## 8. Funcionalidade JavaScript

Foi implementado um botão que permite mostrar e ocultar o contato do barbeiro.

---

## 9. Uso de Bootstrap

Foi utilizado para grid responsivo, navbar e botões.

---

## 10. Testes realizados

* teste em celular e desktop
* verificação de links
* funcionamento do botão
